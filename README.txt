Voici un projet pour le cours sur Git et Github sur OpenClassRooms.
Le but est de mettre en pratique ce qu'il a été appris. Pour ce faire, nous allons créer au moins 2 fichiers et 4 commits en prenant garde de bien les détailler.
Une fois cette première partie effectuée, nous allons envoyer notre travail sur Github.
Enfin, nous renseignerons l'adresse de notre repository en complétant le document texte et l'enverrons dans un ZIP pour correction par les pairs sur OpenClassRooms.
